## 一、系统概述

### 1.1 系统功能

本系统是为「如视海外2025年庞加莱线上内购会」打造的邀请函管理平台，核心功能包括：

| 功能模块 | 说明 |
|---------|------|
| 🎫 邀请函创建 | 销售快速创建多语言邀请函，支持10+模板 |
| 🌍 多语言支持 | 支持 16 种语言，包括阿拉伯语 RTL 布局 |
| 📊 数据追踪 | 实时追踪邀请函状态（待处理/已打开/已接受/已拒绝） |
| 🔔 企微通知 | 客户响应后自动 @销售 通知 |
| 👥 账号管理 | 超管管理销售账号、模板和系统配置 |

### 1.2 用户角色

| 角色 | 权限 | 访问入口 |
|------|------|----------|
| **超管 (ADMIN)** | 系统配置、销售管理、模板管理、数据导出 | `/admin` |
| **销售 (SALES)** | 创建邀请函、查看自己的邀请记录 | `/dashboard` |
| **客户** | 查看邀请函、接受/拒绝邀请 | `/invite/[uniqueToken]` |

## 二、快速开始

### 2.1 访问系统

| 环境 | 地址 |
|------|------|
| 生产环境 | `https://your-domain.vercel.app` |
| 本地开发 | `http://localhost:3000` |

### 2.2 登录

1. 访问系统首页
2. 点击 **"销售登录入口"** 按钮（位于页面底部）

## 三、超管操作指南

### 3.1 进入超管后台

登录后，如果您的账号是超管角色，顶部导航会显示 **"超管后台"** 入口，点击进入。

超管后台地址：`/admin`

### 3.2 数据总览

超管首页显示关键数据指标：

- 📊 总邀请函数量
- ✅ 已接受数量
- ❌ 已拒绝数量
- 👁️ 总打开次数
- 📈 接受率/拒绝率

### 3.3 销售账号管理

路径：`/admin/users`

#### 创建销售账号

1. 点击 **"创建销售账号"** 按钮
2. 填写表单：
   - **销售代号**（用户名，如 `S001`）：必填，用于登录
   - **显示名称**（如 `张三`）：选填，用于界面显示
   - **企业微信ID**：选填，用于 @提及通知
3. 点击 **"创建"**
4. 系统自动生成随机密码
5. **重要**：复制并保存账号信息，密码仅显示一次！

#### 管理账号

- **禁用账号**：点击账号行的「禁用」按钮
- **重置密码**：点击「重置密码」，系统生成新密码

### 3.4 模板管理

路径：`/admin/templates`

- 查看所有可用的邀请函模板
- 启用/禁用特定模板
- 预览不同语言下的模板效果

### 3.5 系统配置

在超管首页可配置：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **活动时间** | 北京时间，ISO 8601 格式 | `2025-06-15T14:30:00+08:00` |
| **活动结束时间** | 活动结束时间 | `2025-06-15T16:00:00+08:00` |
| **内购会链接** | 活动参与链接 | `https://meeting.example.com/join` |
| **企微 Webhook** | 企业微信机器人 Webhook URL | `https://qyapi.weixin.qq.com/...` |

### 3.6 数据导出

路径：`/admin/export`

支持导出邀请函数据为 CSV 格式，包含：
- 客户姓名
- 邀请函状态
- 创建时间
- 访问次数
- 接受/拒绝时间

## 四、销售操作指南

### 4.1 进入销售后台

登录后自动跳转到销售仪表盘：`/dashboard`

### 4.2 创建邀请函

路径：`/dashboard/create`

#### 步骤：

1. **选择模板**：从可用模板中选择一个风格
2. **选择语言**：为客户选择邀请函显示语言
3. **填写客户信息**：
   - **客户姓名**（必填）：将显示在邀请函称呼中
   - **销售备注**（选填）：仅自己可见的备注信息
4. **预览确认**：可切换语言实时预览效果
5. **创建邀请函**：点击「创建」生成唯一链接

#### 创建成功后：

- 获得唯一邀请链接：`https://your-domain.com/invite/xxxxxx`
- 可下载邀请函二维码
- 可一键复制链接分享给客户

### 4.3 邀请函管理

路径：`/dashboard/invitations`

查看所有已创建的邀请函：

| 列 | 说明 |
|----|------|
| 客户姓名 | 邀请函接收人 |
| 状态 | 待处理 / 已打开 / 已接受 / 已拒绝 |
| 语言 | 邀请函显示语言 |
| 访问次数 | 客户打开邀请函次数 |
| 创建时间 | 邀请函创建时间 |
| 操作 | 复制链接 / 下载二维码 / 查看详情 |

#### 状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| PENDING | ⏳ | 待处理，客户尚未访问 |
| OPENED | 👁️ | 已打开，客户已查看但未响应 |
| ACCEPTED | ✅ | 已接受，客户确认参加 |
| DECLINED | ❌ | 已拒绝，客户暂无法参加 |

### 4.4 数据看板

仪表盘首页显示：

- 📊 我的邀请函总数
- ✅ 已接受数量
- ❌ 已拒绝数量
- 📈 接受率统计

## 五、客户邀请函体验

### 5.1 访问邀请函

客户通过销售分享的链接或二维码访问邀请函页面。

### 5.2 邀请函内容

邀请函页面包含：

- 🏷️ 品牌标识和活动主题
- 👤 个性化称呼（根据语言格式化）
- 📅 活动时间（自动转换为客户本地时间）
- 🎁 专属福利说明
- ✨ 行动按钮

### 5.3 客户操作

#### 接受邀请

1. 点击 **"接受邀请"** 按钮
2. 系统生成专属邀请码（如 `RS-2025-ABCDEF`）
3. 显示内购会参与链接
4. 自动通知对应销售

#### 拒绝邀请

1. 点击 **"暂时无法参加"** 按钮
2. 显示遗憾提示
3. 提供 **"重新考虑"** 选项
4. 自动通知对应销售

#### 重新考虑

如果客户改变主意，可点击「重新考虑」恢复到初始状态。

### 5.4 时间显示

- 活动时间自动转换为客户本地时间
- 显示客户所在时区信息
- 支持夏令时自动处理

## 六、常见问题

### Q1: 忘记密码怎么办？

联系超管重置密码。超管可在「销售管理」页面为您重置。

### Q2: 邀请函链接失效？

邀请函链接永久有效，不会过期。如果无法访问，请检查：
- 网络连接是否正常
- 链接是否复制完整

### Q3: 客户收不到邀请函？

邀请函是网页链接形式，请通过以下方式分享：
- 直接发送链接
- 发送二维码图片
- 通过邮件/微信/WhatsApp 等渠道分享

### Q4: 如何修改邀请函内容？

创建后的邀请函无法修改。如需更改，请创建新的邀请函。

### Q5: 客户接受后如何参加活动？

客户接受邀请后会看到：
1. 专属邀请码
2. 内购会参与链接

请提醒客户保存这些信息。

### Q6: 企微通知没有收到？

请检查：
1. 超管是否正确配置了企微 Webhook
2. 销售账号是否填写了正确的企业微信 ID
3. 企微机器人是否在正确的群聊中

### Q7: 阿拉伯语显示不正确？

阿拉伯语使用从右向左（RTL）布局，系统已自动处理。如果显示异常，请尝试刷新页面或使用 Chrome 浏览器。
